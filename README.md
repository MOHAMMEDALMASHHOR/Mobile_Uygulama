# Ders Görev Takip Uygulaması (Task Tracker App)

## Project Overview

This project is a React Native mobile application built as a task management tool specifically designed for mobile application development coursework. It demonstrates core concepts of state management, user interaction, and dynamic list rendering without relying on external UI libraries.

## Core Features

*   **Task Management:** Users can add tasks with specific titles, descriptions, and deadlines.
*   **Validation:** Built-in checks prevent the creation of empty tasks using native `Alert` components.
*   **Categorization & Priority:** Tasks can be assigned specific categories (e.g., Mobile, Design) and priority levels (Low, High).
*   **Dynamic Statistics:** The dashboard automatically calculates and displays the total, active, and completed task counts in real-time.
*   **Filtering System:** Users can seamlessly filter the task list view by "All", "Active", or "Completed" statuses.
*   **Inline Styling:** The UI is crafted using clean, highly readable inline styles to ensure a lightweight and easily maintainable codebase.

## Setup & Installation

1.  **Clone the repository:** `git clone https://github.com/MOHAMMEDALMASHHOR/Mobile_Uygulama.git`
2.  **Install dependencies:** `npm install`
3.  **Start the Expo server:** `npx expo start`
4.  **Run on device:** Scan the generated QR code with the Expo Go app on your iOS or Android device.
